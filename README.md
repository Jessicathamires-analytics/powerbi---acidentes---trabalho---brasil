## Análise de Acidentes de Trabalho - Brasil (Dados Fictícios)

### Objetivo
Analisar acidentes de trabalho em um país fictício, considerando regiões, tipos de acidentes e ocorrências de fatalidades.

### Ferramentas Utilizadas 
- Power BI
- Excel
- Power Query
- DAX (básico)

### Análises Realizadas 
- Acidentes por região
- Tipos de acidentes mais frequentes
- Comparação entre acidentes fatais e não fatais

### Dasjboard 
O dashboard foi desenvolvido no Power BI Desktop e está disponível no arquivo Projeto I.pbix neste repositório.


### Observação
Projeto desenvolvido com dados fictícios para fins educacionais.
